Today

The KYC (Know Your Customer) process utilises a broken model that relies on manual data provisioning, human intervention, antiquated, siloed and still largely paper-based systems, making the process complex, inefficient, risky and exclusive.

Tomorrow

Built on DLT

KYC Asset is built on distributed ledger technology, allowing participants to agree on a shared source of truth and perception using execution of smart contracts over secure, immutable data. The value and security of a network built on DLT grows exponentially with the number of participants. Existing systems can also be integrated easily and solutions can be tailored to meet specific user and client requirements.

Enterprise Ready

KYC Asset represents a utility to all participants but is built from the client's perspective and provides enterprise level integration into financial institutions and corporate entities as well as legal and data verification services. Clients are able to control and manage their own data, platform and client relationships, whilst allowing financial institutions to manage risk assessment and compliance.

Built for Interoperability

KYC Asset is built on an open data model which provides interoperability and scalability of data being shared on a distributed ledger. By building on a network that is owned by participants and not by a single trusted party, participants are able to build their own solutions and integrations utilising the same underlying data, therefore incentivising participants to utilise the KYC Asset utility.

Automated Verification

KYC Asset leverages an event driven model. State transitions on the ledger raise events which can trigger processes within integrated systems to provide automation over digital claims and data representing legal entity identification.
